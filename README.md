# 微信拼豆生成器

一个基于微信小程序的拼豆参考图生成工具，可将本地图片转换为适合查看和制作的拼豆效果预览与参考图。

## 功能特性

- 支持选择本地图片生成拼豆图
- 针对人物图片做了更友好的颜色映射和细节保留
- 提供方块风格的效果预览
- 提供居中排布、带留白边距的参考图
- 纯前端实现，无需后端服务

## 使用流程

1. 选择一张本地图片。
2. 生成拼豆结果。
3. 查看效果预览。
4. 查看或导出参考图。

## 输出说明

- 效果预览：用于快速查看拼豆化后的整体效果，主体更紧凑。
- 参考图：用于实际照图制作，主体会嵌入到居中的参考板布局中。

## 本地运行

1. 安装依赖：`npm install`
2. 打开微信开发者工具。
3. 选择“导入项目”，目录指向当前仓库根目录。
4. 使用项目现有配置完成本地预览运行。

## 测试

运行以下命令执行测试：

```bash
npm test
```

## 项目结构

```text
pages/      页面入口与结果页
utils/      图像处理、配色、清理与渲染逻辑
tests/      单元测试与页面测试
openspec/   需求变更与规格文档
```

## 说明与限制

- 项目当前无需后端服务。
- 生成效果会受到原始图片质量影响。
- 当前结果页仅保留两种输出：效果预览、参考图。
